## 跨领域综合分析（3题）

### Q5-L5-01
- **问题**：2023年各省有几十个跨领域指标（经济、就业、教育、医疗、环境、人口、交通等），人类很难直接看懂这么多维度的数据。请用数据自身的方法把这些指标浓缩成2-3个核心维度，帮我解释每个维度代表什么（比如"经济活力""民生保障""环境质量"），然后用这些核心维度给各省的综合发展水平打分排名。对比一下这样算出来的排名和"简单平均"算出来的排名有什么不同。
- **涉及表**：`economic_indicator_data`, `employment_data`, `edu_data`, `medical_health_data`, `env_monitor_data`, `population_data`, `admin_region_data`
- **分析类别**：跨领域综合分析（全域降维综合评价）
- **所需算法**：PCA 主成分分析（sklearn.decomposition.PCA）
- **考察点**：主成分命名解释、降维后排名与原指标直接排名的差异

### Q5-L5-02
- **问题**：经济、环境、社会系统之间的关系很复杂，不是简单的"线性"关系能描述的。2023年，请帮我分析以下几组关系，看看它们之间有没有复杂的、非线性的信息重叠：
  - 经济增长（GDP增长率）和空气质量（PM2.5）
  - 教育投入（教育支出）和教育成果（识字率）
  - 医疗资源和人口老龄化（千人床位数和老龄化率）
  - 投资和就业（固定资产投资增长率和失业率）
  对比"简单相关分析"和"复杂关联分析"两种方法的结果差异——如果差异很大，说明这些关系可能不是简单的直线关系。
- **涉及表**：`economic_indicator_data`, `env_monitor_data`, `edu_data`, `medical_health_data`, `population_data`, `employment_data`, `admin_region_data`
- **分析类别**：跨领域综合分析（复杂系统信息分析）
- **所需算法**：Mutual Information（互信息）
- **考察点**：连续变量离散化、MI vs Pearson 对比解读

### Q5-L5-03
- **问题**：一个省的发展健康与否，不能只看经济——要看经济、民生、教育、环境、人口结构五个系统是否协调。请用系统协调度模型，评估2023年各省五个系统的协调发展水平。我需要知道：
  1. 哪些省份五个方面都发展得不错（整体协调度最高）？
  2. 哪些省份有明显的"短板"（某个方面严重落后）？
  3. 哪些省份各方面发展最均衡（差距最小）？
- **涉及表**：`economic_indicator_data`, `employment_data`, `medical_health_data`, `edu_data`, `env_monitor_data`, `population_data`, `admin_region_data`
- **分析类别**：跨领域综合分析（多系统协调度）
- **所需算法**：多维 CCD 扩展模型（n 系统通用公式）
- **考察点**：n>2 的 CCD 推广、短板效应识别、均衡性判断

## 时间序列与趋势分析（3题）

### Q5-L5-04
- **问题**：2020年到2023年全国季度GDP数据中，既有长期增长趋势，又有短期波动（比如疫情冲击、政策刺激等）。我想把"趋势"和"波动"分开：先去掉趋势，只看波动部分，然后找出波动特别大的季度（无论正向还是负向）。最后结合重大政策或事件（如疫情、经济刺激计划）来分析每个异常波动的原因。
- **涉及表**：`economic_indicator_data`
- **分析类别**：时间序列与趋势分析
- **所需算法**：SMA 去趋势 + Z-score + IQR 双模异常检测 + 并集融合
- **考察点**：去趋势的窗口选择、多方法异常融合、异常的业务归因

### Q5-L5-05
- **问题**：房地产投资和经济增长之间的关系很复杂——是房地产带动了经济，还是经济好了房地产才热？我想用多种方法，从不同角度分析两者的关系：
  - 它们在同一时期的相关性如何？
  - 排除极端值的影响后，关系还稳不稳固？
  - 房地产投资是否提前于GDP变化（比如提前半年到一年）？
  综合所有分析，判断房地产投资是经济的"领先指标""同步指标"还是"滞后指标"？
- **涉及表**：`real_estate_data`, `economic_indicator_data`, `admin_region_data`
- **分析类别**：时间序列与趋势分析
- **所需算法**：Pearson + Spearman + TLCC 三模联合分析
- **考察点**：相关 vs 滞后相关 vs 秩相关的互补解读

### Q5-L5-06
- **问题**：都说经济增长能带动就业，但这个"带动能力"这些年是在增强还是减弱？我想算算每年GDP每增长1%，就业人口增长百分之几（就业弹性），然后看这个弹性值的变化趋势。如果弹性越来越低，说明"增长不带动就业"的问题确实存在——经济在增长但就业没跟上。弹性是否在某一年发生了明显转折？
- **涉及表**：`employment_data`, `economic_indicator_data`
- **分析类别**：时间序列与趋势分析
- **所需算法**：弹性系数 + 线性趋势 + CMA（Centered Moving Average）
- **考察点**：弹性系数的趋势解读、结构突变的业务含义

## 排名与比较（3题）

### Q5-L5-07
- **问题**：我想对2023年各省的综合实力做一个全面的排名——不是简单看GDP，而是分两步：第一步，在经济、民生、教育、环境、交通、人口结构这六个领域分别打分（每个领域选2个核心指标，计算百分位排名均值作为领域分）；第二步，用数据自身说话来确定六个领域的权重（不需要人工设定），再算出综合排名。最后对比一下：用数据定权重排出来的名次，和"六个领域等权重"排出来的名次有多大差异？哪些省份因为权重调整名次发生了明显变化？
- **涉及表**：全部10张表
- **分析类别**：排名与比较（两级综合排名）
- **所需算法**：PERCENT_RANK + 熵权法 + 两层聚合
- **考察点**：两级排名体系设计、熵权 vs 等权的差异分析

### Q5-L5-08
- **问题**：我想评估2023年各省的"营商环境"——看企业活跃度（每万人企业数、私营企业占比）、交通便利度（货运量、客运量）、经济基础（人均GDP、固定资产投资）和人力资源（大学生数、识字率）四个维度，用数据自动定权重给各省排名。然后把"营商环境排名"和"GDP排名"对比，看看哪些省份的营商环境排名远高于其经济排名（说明制度环境好但经济潜力还没完全释放），哪些省份刚好相反。
- **涉及表**：`enterprise_data`, `transport_data`, `economic_indicator_data`, `edu_data`, `population_data`, `admin_region_data`
- **分析类别**：排名与比较
- **所需算法**：熵权-TOPSIS、排名差分析
- **考察点**：专题排名 vs 总量排名的偏差解释

### Q5-L5-09
- **问题**：2023年，如果以综合实力最强的省份（排名第一）为"标杆"，其他各省与标杆的差距到底有多大？对人均GDP、识字率、千人床位数、森林覆盖率、失业率、老龄化率（后两个反向）这6个指标，分别算算每个省与标杆省的差距百分比，然后汇总成一个"综合差距分"。哪些省份离标杆最近（差距<10%）？哪些省份差距最大（差距>50%）？各自的短板在哪里？
- **涉及表**：`economic_indicator_data`, `edu_data`, `medical_health_data`, `env_monitor_data`, `employment_data`, `population_data`, `admin_region_data`
- **分析类别**：排名与比较
- **所需算法**：标杆偏离度分析（Benchmark Distance）
- **考察点**：标杆管理思想的量化实现

## 多维度深度分析（3题）

### Q5-L5-10
- **问题**：从"区域（东部/中部/西部/东北）× 时间（2021/2022/2023年）× 经济指标（GDP/财政收入/固定资产投资）"三个维度做一个全面的交叉分析。我需要看到：每个区域每年的GDP变化趋势、每个区域每年的"财政收入/GDP"比例（经济含金量）、以及哪些年份哪些区域的投资增长异常快。这样能全面了解我国区域经济的三维画像。
- **涉及表**：`economic_indicator_data`, `admin_region_data`
- **分析类别**：多维度深度分析
- **所需算法**：CUBE（三维交叉汇总）
- **考察点**：三维 CUBE 设计与结果解读

### Q5-L5-11

  - "全面优秀"的省份：经济水平高 + 民生好 + 环境优
  - "全面堪忧"的省份：经济水平低 + 民生差 + 环境差
  同时看看在每种组合中各有多少个省份。这能告诉我：我国有多少省份实现了"高质量发展"，有多少还在"粗放发展"？
- **涉及表**：`economic_indicator_data`, `medical_health_data`, `env_monitor_data`, `employment_data`, `admin_region_data`

- **所需算法**：NTILE(4) × 3 维交叉分组

### Q5-L5-12

## 政策评估与关联分析（3题）

### Q5-L5-13
- **问题**："教育投入能推动经济增长"这个说法流传很广，但我想用多种方法层层验证：
  - 第一层：简单看教育支出增长率和GDP增长率之间的弹性关系
  - 第二层：用多年的数据建一个模型，把投资、人口等因素都控制住，看教育投入的独立贡献
  - 第三层：用季度数据检验"先有教育投入，后有经济增长"这个时间先后顺序是否成立
  三种方法的结果一致吗？如果矛盾，哪个结论更可信？教育到底是不是经济增长的"发动机"？
- **涉及表**：`edu_data`, `economic_indicator_data`, `population_data`, `admin_region_data`
- **分析类别**：政策评估与关联分析（三级验证链）
- **所需算法**：弹性系数 + Panel OLS + Granger 因果检验
- **考察点**：三级证据链的交叉验证、结果一致性判断

### Q5-L5-14
- **问题**：有个说法叫"环境 Kuznets 曲线"——经济发展初期环境会恶化，但过了某个拐点后，经济增长反而会改善环境。我想用数据验证一下这个说法：2020年到2023年，各省的PM2.5和GDP之间的关系是什么？用"脱钩指数"来算：经济增长的同时，污染是在同步增长（未脱钩）、增速放缓（相对脱钩）还是绝对下降（强脱钩）？各省处于什么阶段？产业升级（服务业占比提高）是否有助于实现脱钩？
- **涉及表**：`env_monitor_data`, `economic_indicator_data`, `admin_region_data`
- **分析类别**：政策评估与关联分析（环境-经济脱钩）
- **所需算法**：OECD 脱钩指数（Decoupling Index）
- **考察点**：脱钩状态分类、脱钩与产业结构的关系

### Q5-L5-15
- **问题**：2023年，各省的医疗资源配置到底是"供过于求"还是"供不应求"？我想根据各省的人口规模和老龄化程度，算出一个"理论上应该有多少医疗资源"的合理值，然后对比实际值。如果实际远低于理论值，说明存在"医疗资源供给不足"——这些省份的居民看病可能很困难。哪些省份存在严重的医疗供给不足？这些省份是经济欠发达地区吗？
- **涉及表**：`medical_health_data`, `population_data`, `admin_region_data`
- **分析类别**：政策评估与关联分析（供需匹配）
- **所需算法**：供需残差分析（Supply-Demand Residual）
- **考察点**：回归模型构建、残差的经济解释、资源错配识别

## 异常检测与专项分析（2题）

### Q5-L5-19
- **问题**：我想做一次全面的"发展健康体检"，对2020年到2023年全国各省的GDP增长率和失业率进行全面扫描，找出所有"不对劲"的信号：
  1. 先找出数值本身异常偏大或偏小的年份-省份
  2. 再找出变化幅度异常大的年份-省份（和往年相比突然猛增或猛降）
  3. 综合以上两者，标记"高置信度异常"
  4. 对于这些异常点，从其他表中找找原因——是不是同期投资发生了突变？是不是人口发生了显著变化？
  最终输出一份"异常诊断报告"。
- **涉及表**：`economic_indicator_data`, `employment_data`, `population_data`, `admin_region_data`
- **分析类别**：异常检测与专项分析
- **所需算法**：3σ + 同比突变 + 跨表交叉验证
- **考察点**：多方法异常交集确认、跨表佐证、异常的业务诊断

### Q5-L5-20
- **问题**：请对2023年全国各省进行一次全面扫描，在一份报告中综合输出以下信息：
  1. **极端表现**：任何指标上表现特别好或特别差的省份
  2. **突然变化**：相比往年变化幅度特别大的指标和省份
  3. **资源不够用**：医疗、教育等资源供给不足（低于理论需求）的省份
  4. **系统不协调**：经济-民生-环境之间发展失衡的省份
  5. **"另类"省份**：用聚类方法找出的那些"不像自己那类人"的省份（距离聚类中心最远）
  这5个维度汇总成《2023年各省发展异常特征扫描报告》。
- **涉及表**：全部 10 张表
- **分析类别**：异常检测与专项分析（全链路综合诊断）
- **所需算法**：3σ + IQR + 同比突变 + 供需残差 + CCD + K-means 离群检测
- **考察点**：多维度异常融合、综合诊断报告生成逻辑
